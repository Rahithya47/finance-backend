# 💰 Finance Dashboard Backend System

A robust RESTful API backend for managing financial data with role-based access control.

## 🚀 Features

- **User Authentication**: JWT-based authentication
- **Role-Based Access Control**: Viewer, Analyst, Admin roles
- **Financial Records Management**: CRUD operations for income/expense tracking
- **Dashboard Analytics**: Summary, trends, category breakdowns
- **Input Validation**: Comprehensive request validation
- **Pagination & Filtering**: Efficient data retrieval

## 📋 Tech Stack

- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB with Mongoose
- **Authentication**: JWT (JSON Web Tokens)
- **Validation**: express-validator
